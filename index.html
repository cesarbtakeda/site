  • 7-Zip-CVE-2025-0411-Exploit

    Esta vulnerabilidade (CVSS SCORE 7.0) permite que atacantes remotos contornem o mecanismo de proteção Mark-of-the-Web em instalações afetadas do 7-Zip. A interação do usuário é necessária para explorar essa vulnerabilidade, uma vez que o alvo deve visitar uma página maliciosa ou abrir um arquivo malicioso. O defeito específico existe dentro do manuseio de arquivos compactados. Ao extrair arquivos de um arquivo criado que possui o Mark-of-the-Web, o 7-Zip não propaga o Mark-of-the-Web para os arquivos extraídos. Um atacante pode aproveitar essa vulnerabilidade para executar código arbitrário no contexto do usuário atual.
